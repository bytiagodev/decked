---

Built around a California-inspired skate and lifestyle brand designed from scratch. Bold visuals, warm tones, and a consistent identity carried through every page, from the brand identity to the product copy.

## Tech stack

| Layer | Implementation |
|-------|---------------|
| Frontend | HTML + Vanilla CSS + JavaScript |
| Data | localStorage |
| Hosting | GitHub Pages |

## Product catalogue

Products are organised across decks, trucks, wheels, apparel, hardware, and accessories. All products, descriptions, pricing, and stock levels were written specifically for the project.

## Design

**Typography:** Passion One for headings, Quicksand for body text.

**Colour palette:**

| Colour | Hex | Role |
|--------|-----|------|
| Sky Blue | `#87CEEB` | Nav, headers, links |
| Sandy Gold | `#F7E8C8` | Page background, card fills |
| Coral Red | `#E8614D` | CTAs, prices, badges |
| Seafoam | `#4ECDC4` | Tags, secondary buttons |
| Sunset Orange | `#F5A623` | Highlights, sale tags |
| Concrete | `#8E8E82` | Borders, muted text |
| Cream White | `#FEFDF5` | Surfaces, button text |
| Dark Coal | `#2C2C28` | Headings, footer |

## Features

- User registration and login
- Session management with logout
- Product browsing by category
- Product detail pages
- Shopping cart
- Wishlist
- Order history
- Featured products on homepage
- Stock status (In Stock / Low Stock / Out of Stock)
- Fully responsive
